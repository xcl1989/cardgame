# 经典经营分析大屏布局模板

## 模板概述

这是一个经典的企业经营分析大屏布局模板，适用于展示公司/部门的多维度经营数据。

**特点**：
- 三列布局 + 底部一行
- 中央轮播图为视觉焦点
- 左右两侧对称分布
- 顶部指标卡展示关键数据
- 底部密集展示详细分析

**适用场景**：
- 公司经营分析
- 部门业绩展示
- 项目数据看板
- 财务数据驾驶舱

## 典型组件组合模式

### 模式 1：卡片组合（左侧列，重复 4 次）

```bash
# 装饰条
python3 scripts/add_component.py \
  --screen-id <ID> \
  --chart-type decorate \
  --top 92 --left 16 \
  --width 412 --height 29 \
  --view-name "bar" && \

# 文本标签（叠加在装饰条上）
python3 scripts/add_component.py \
  --screen-id <ID> \
  --chart-type text \
  --top 100 --left 62 \
  --width 176 --height 35 \
  --view-name "卡片标题" && \

# 卡片容器
python3 scripts/add_component.py \
  --screen-id <ID> \
  --chart-type card \
  --top 120 --left 16 \
  --width 412 --height 196 \
  --view-name "卡片 1"
```

**坐标规律**：
- 装饰条：`top = 92, 341, 590, 838`（间距 249）
- 卡片：`top = 120, 369, 618, 866`（间距 249）
- 文本：`top = 装饰条 top + 8`, `left = 62`

## 创建顺序建议

### 推荐步骤

1. **头部区域** (top: 0-80)
   - 添加头部装饰
   - 添加大屏标题文本

2. **顶部指标区** (top: 80-200)
   - 添加 4 个方形装饰
   - 添加 4 个图标
   - 添加 6 个指标卡
   - 添加右侧排名图

3. **中央轮播图** (top: 200-300)
   - 添加轮播图（视觉焦点）

4. **左侧卡片列** (top: 300-830)
   - 循环 4 次添加：装饰条 → 文本 → 卡片

5. **右侧图表区** (top: 100-800)
   - 添加线图
   - 添加双轴图

6. **底部图表行** (top: 830-1080)
   - 添加 4 组装饰条 + 文本
   - 添加饼图、仪表盘、双轴图、漏斗图

### 图层顺序注意事项

1. **先添加背景层**：头部装饰、方形装饰
2. **再添加容器层**：卡片、装饰条
3. **然后添加内容层**：文本标签、图标
4. **最后添加图表层**：各种图表组件

**重要**：添加组件时必须串行执行（使用 `&&`），避免并行导致配置覆盖！

## 使用示例

### 创建基础框架

```bash
SCREEN_ID=3654239893
MODEL_ID=2597080602
DIM_FIELD=projectNo
IND_FIELD=receivingAmount

# 1. 头部
python3 scripts/add_component.py \
  --screen-id $SCREEN_ID \
  --model-id $MODEL_ID \
  --dimension-field $DIM_FIELD \
  --indicator-field $IND_FIELD \
  --chart-type head \
  --top 0 --left 0 \
  --width 1920 --height 80 \
  --view-name "头部 1" && \

# 2. 大屏标题
python3 scripts/add_component.py \
  --screen-id $SCREEN_ID \
  --model-id $MODEL_ID \
  --dimension-field $DIM_FIELD \
  --indicator-field $IND_FIELD \
  --chart-type text \
  --top 10 --left 780 \
  --width 385 --height 60 \
  --view-name "公司经营分析大屏"
```

### 创建左侧卡片列

```bash
# 循环添加 4 组卡片
for i in 1 2 3 4; do
  TOP_DECORATE=$((92 + (i-1)*249))
  TOP_TEXT=$((100 + (i-1)*249))
  TOP_CARD=$((120 + (i-1)*249))
  
  python3 scripts/add_component.py \
    --screen-id $SCREEN_ID \
    --model-id $MODEL_ID \
    --dimension-field $DIM_FIELD \
    --indicator-field $IND_FIELD \
    --chart-type decorate \
    --top $TOP_DECORATE --left 16 \
    --width 412 --height 29 \
    --view-name "bar" && \
  
  python3 scripts/add_component.py \
    --screen-id $SCREEN_ID \
    --model-id $MODEL_ID \
    --dimension-field $DIM_FIELD \
    --indicator-field $IND_FIELD \
    --chart-type text \
    --top $TOP_TEXT --left 62 \
    --width 176 --height 35 \
    --view-name "卡片$i" && \
  
  python3 scripts/add_component.py \
    --screen-id $SCREEN_ID \
    --model-id $MODEL_ID \
    --dimension-field $DIM_FIELD \
    --indicator-field $IND_FIELD \
    --chart-type card \
    --top $TOP_CARD --left 16 \
    --width 412 --height 196 \
    --view-name "卡片$i"
done
```

## 变体布局

### 简化版（无轮播图）

- 移除中央轮播图
- 左侧卡片列扩展至中央
- 右侧图表区增加图表数量

### 增强版（更多指标卡）

- 顶部指标卡从 6 个增加到 8 个
- 调整指标卡宽度：从 171 减至 150
- 保持总宽度不变

### 紧凑版（更小画布）

- 画布调整为 1600×900
- 所有组件尺寸按比例缩小（0.83 倍）
- 保持布局结构不变

## 注意事项

### 1. 组件添加顺序
- 必须先添加背景/装饰组件
- 再添加容器组件
- 最后添加内容组件
- **串行执行**，避免并行覆盖

### 2. 坐标计算
- 使用变量存储坐标值
- 保持间距一致
- 注意组件间的叠加关系

### 3. 尺寸调整
- 调整卡片宽度时，同步调整装饰条宽度
- 调整卡片高度时，注意影响下方组件
- 保持整体布局平衡

### 4. 颜色搭配
- 头部与装饰条颜色协调
- 指标卡与卡片背景色区分
- 图表颜色与主题一致
