# 大屏布局规划模板

## 画布规格

**标准画布**: 1920 × 1080 像素

## 组件推荐尺寸

| 组件类型 | 推荐宽度 | 推荐高度 | 说明 |
|---------|---------|---------|------|
| 指标卡 | 220-240 | 120-140 | 顶部展示，高度要足够显示数值 |
| 主图表（中间） | 800-900 | 520-580 | 视觉焦点，占据画布中心 |
| 侧边图表 | 480-520 | 260-290 | 左右两侧，对称分布 |
| 底部图表 | 580-620 | 280-320 | 底部并排，通常 3 个 |
| 饼图 | 370-520 | 270-290 | 根据空间调整 |

## 坐标计算公式

### 指标卡排布

```python
# 参数
card_width = 230  # 指标卡宽度
card_height = 130  # 指标卡高度
gap = 10  # 间距
top = 40  # 顶部距离
start_left = 20  # 起始左边距

# 计算第 n 个指标卡的位置（从 1 开始）
n = 1
left = start_left + (n - 1) * (card_width + gap)
# 第 1 个：left = 20
# 第 2 个：left = 260
# 第 3 个：left = 500
# ...
```

### 底部图表均分

```python
# 参数
canvas_width = 1920
chart_count = 3  # 3 个图表
gap = 20  # 间距
start_left = 20

# 计算每个图表宽度
chart_width = (canvas_width - start_left * 2 - gap * (chart_count - 1)) // chart_count
# chart_width = (1920 - 40 - 40) / 3 = 613

# 计算第 n 个图表的位置（从 1 开始）
n = 1
left = start_left + (n - 1) * (chart_width + gap)
# 第 1 个：left = 20
# 第 2 个：left = 653
# 第 3 个：left = 1286
```

## 布局规划工具

### Excel 模板

创建 Excel 表格，包含以下列：

| 序号 | 组件名称 | 类型 | 宽度 | 高度 | top | left | 数据模型 | 维度字段 | 指标字段 |
|------|---------|------|------|------|-----|------|----------|----------|----------|

| 2 | 月度收款 | 柱图 | 510 | 260 | 190 | 20 | 收款登记 | receivingDate | receivingAmount |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |

### Python 脚本辅助计算

```python
# 计算指标卡位置
def calc_metric_positions(count=8, width=230, height=130, gap=10, top=40, start_left=20):
    positions = []
    for i in range(count):
        left = start_left + i * (width + gap)
        positions.append({
            'top': top,
            'left': left,
            'width': width,
            'height': height
        })
    return positions

# 计算底部图表位置
def calc_bottom_positions(count=3, top=760, height=300, canvas_width=1920, start_left=20, gap=20):
    chart_width = (canvas_width - start_left * 2 - gap * (count - 1)) // count
    positions = []
    for i in range(count):
        left = start_left + i * (chart_width + gap)
        positions.append({
            'top': top,
            'left': left,
            'width': chart_width,
            'height': height
        })
    return positions
```

## 注意事项

1. **指标卡高度**: 至少 120px，推荐 130-140px，确保数值完整显示
2. **组件间距**: 至少 10px，推荐 15-20px，避免视觉拥挤
3. **主图表尺寸**: 至少 800×500，突出视觉焦点
4. **组件数量**: 1920×1080 画布建议 12-18 个组件
5. **颜色搭配**: 同类型图表使用相近颜色，不同类型使用对比色
6. **数据更新**: 确保选择的模型有实时数据，避免显示"暂无数据"
