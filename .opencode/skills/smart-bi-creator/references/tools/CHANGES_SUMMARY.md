## 🔧 技术细节

### rename_component.py 实现原理

```python
# 同时修改 3 个位置
def rename_component(screen_id, uuid, new_title):
    # 1. 获取大屏配置
    config = get_screen_config(screen_id)
    
    # 2. 修改 layer.title
    for layer in config['config']['layers']:
        if layer['uuid'] == uuid:
            layer['title'] = new_title
    
    # 3. 修改 componentMap.viewName
    config['componentMap'][uuid]['viewName'] = new_title
    
    # 4. 修改 layout.title
    for layout in config['layouts']:
        if layout['uuid'] == uuid:
            layout['title'] = new_title
    
    # 5. 提交更新
    update_screen_config(screen_id, config)
```

**关键点**: 必须同时修改 3 个位置，缺少任何一个都会导致显示问题

### 1. 布局规划
```

中部：主图表 (1100×500, 居中)
底部：4 个图表 (460×300, 均匀分布)
```

### 2. 安全间距
- 组件之间：≥20px
- 指标卡高度：≥140px
- 主图表宽度：1100px（1920 画布）

### 3. 调试流程
1. 使用 `--show-info` 查看布局
