# News Fetcher 使用参考

## 使用task工具的完整流程

### 步骤1: 读取配置文件

首先读取 `assets/config.json` 获取新闻源列表。

### 步骤2: 为每个新闻源启动子代理

使用以下模式为每个网站启动explore子代理:

```
task(
    description="查询[网站名称]新闻",
    prompt="请访问 [网站URL] 并查询今天的重大新闻。提取最重要的10-15条新闻标题和简要内容，按重要性排序。返回格式为清晰的新闻列表。",
    subagent_type="explore"
)
```

### 步骤3: 汇总结果

将所有子代理返回的新闻汇总，去重后按类别和重要性排序输出。

## 输出模板

最终输出应包含:

1. 每个新闻源的独立结果
2. 综合热点TOP 10
3. 分类新闻汇总（政治、经济、国际、社会、科技、体育等）

## 配置文件修改示例

添加新的新闻源:

```json
{
  "name": "新浪新闻",
  "url": "https://news.sina.com.cn"
}
```

修改默认新闻数量:

```json
"default_news_count": 20
```
