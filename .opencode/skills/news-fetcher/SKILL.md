# News Fetcher Skill - 新闻获取技能

## 概述

这个技能用于从多个新闻网站获取今日重大新闻，支持灵活的网站配置和并行查询，为用户提供全面的新闻资讯汇总。

## 功能特点

- 📰 **多源新闻聚合** - 支持百度、网易、新浪等主流新闻平台
- ⚡ **并行高效获取** - 使用子代理并行查询，提高效率
- 🔧 **灵活配置** - 可自定义新闻源和获取数量
- 📊 **智能分类汇总** - 按热点、国际、经济、科技等分类整理

## 支持的新闻源

| 新闻源 | 网址 | 特点 |
|---------|-------|------|
| 百度新闻 | [https://news.baidu.com | 海量中文资讯，热点榜单 |
| 网易新闻 | [https://news.163.com | 深度报道，有态度新闻 |
| 新浪新闻 | [https://news.sina.com.cn | 实时滚动，微博热点 |

## 配置文件

配置文件位于 `assets/config.json`，包含要查询的新闻网站列表。用户可以修改此文件来添加或删除新闻源。

当前配置：
```json
{
  "sources": [
    {
      "name": "百度新闻",
      "url": "https://news.baidu.com"
    },
    {
      "name": "网易新闻",
      "url": "https://news.163.com"
    },
    {
      "name": "新浪新闻",
      "url": "https://news.sina.com.cn"
    }
  ],
  "default_news_count": 15,
  "max_news_per_source": 15,
  "enable_parallel_fetch": true
}
```

## 使用方法

### 快速开始

直接询问"今天有什么新闻"即可触发技能，自动从配置的新闻源获取并汇总新闻。

### 详细流程

使用task工具并行查询，为每个新闻源启动独立的子代理进行并行查询：

1. 读取配置文件获取网站列表
2. 对每个网站使用task工具启动explore子代理
3. 汇总所有子代理的结果
4. 在子代理中，使用webfetch获取新闻内容，并提取新闻标题和摘要
5. 按重要性排序输出

**注意**：在子代理中，不要使用write工具写入数据，因为write工具无法在子代理中使用。

## 输出格式

新闻汇总将包含：

### 1. 综合热点 TOP 10
精选最重要的 10 条新闻，每条包含：
- 新闻标题
- 简要内容
- 来源链接（使用 Markdown 格式，让链接在新页签打开）

**链接格式要求**：所有新闻链接应在新页签打开，使用以下格式：
```markdown
[查看详情](URL)
```

### 2. 分类新闻
按以下类别组织新闻：
- 🌍 国际新闻
- 📈 经济财经
- 🤖 科技前沿
- 🎭 社会热点
- ⚽ 体育资讯

### 3. 新闻源链接
提供各新闻源的首页链接，方便用户直接访问

## 新闻源扩展

要添加新的新闻源，编辑 `assets/config.json`，在sources数组中添加新对象：
```json
{
  "name": "新闻源名称",
  "url": "新闻网站URL"
}
```

**推荐添加的新闻源**：
